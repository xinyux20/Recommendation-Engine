# Movie Recommendation System

This project builds an AI-powered recommendation system using the TMDB movie dataset.

## Features
- Content-based filtering using movie descriptions
- Similarity matching using TF-IDF embeddings
- Explanation generation for recommendations

## Dataset
TMDB 5000 Movies dataset

## Example
If a user likes "Inception", the system recommends movies with similar descriptions and genres.

## Evaluation
- Precision@K
- Recall@K
- Diversity
